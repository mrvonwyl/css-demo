Box Model, Marging, Padding, Box-Sizing

info: w3schools box sizing

box-sizing default: content-box
margin default: varies (mostly 0)
padding default: varies (mostly 0)

General Info

  • margin, border, padding, content
  • margin and padding property shortcuts:
    • margin-top: top, margin-right: right
    • margin: top right bottom left
    • margin: top/bottom right/left
    • margin: top right/left bottom
  • margins of siblings are overlapping, the bigger value is relevant (e.g. margin-bottom of the green box and margin-top of the purple box)

content-box

width: 100%;
width: calc(100% - 42px);
  • width and height properties affect only the content

border-box

width: 100%;
width: calc(100% - 20px);
  • width and height properties affect the content including padding and border
